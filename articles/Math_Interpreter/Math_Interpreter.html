Math Interpreter

This is a stack based math interpreter I made for personal enjoyment. Regrettably, the code is a messy but I don't have time to go back and clean it up.

I made it mostly to educate myself in the ways of stack based math and reverse-polish-notation. Maybe someday I can re-use this in another project.

The interpreter works as follows:

  1. User enters equation.
  2. Text is parsed into symbols.
  3. Symbols are rearranged into reverse-polish-notation... aka "postfix".
  4. Symbols are then put onto the stack as the final equation.
  5. Equation can then be calculated quickly and efficiently.

As well as the basic operators like + - / * ^ ... sin(), cos(), tan(), log(), abs(), etc. are also available.
