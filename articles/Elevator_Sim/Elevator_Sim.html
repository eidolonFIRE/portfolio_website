Elevator Simulation

I was reading Don Norman's The Design of Everyday Things and he mentioned the superiority of "destination" based elevators over the traditional type (traditional having floor selection inside the elevator). I was immediately inspired to write a framework for simulating elevator strategies and comparing them.

Above, you can see a simulation being rendered for 3 elevators and 8 floors. Various metrics are shown live for each floor such as: call buttons, who's waiting for what floor, which elevator is stopping there, etc.

Using python multiprocessing, I am able to run a large batch job to get data for all building configurations. Each configuration is run dozens of times to get accurate statistical results.

In a future re-visit of this project, I will add a destination based strategy to the simulation and compare the results. In the mean time, source files can be found on Elevator_Sim GitHub. Don't judge it too hashly as I hammered this project out over the course of a few evenings.
