LED Tree

No going back to regular tree lights after these...

Source Code on Github!

The lights are individually addressable led strips. They work by feeding the data to the first led and the data is pushed from one led to the next all the way down the chain using shift registers. I'm using a raspberry pi A+ for the controller because it's what I had on hand.

The pi is running a websocket server and listening for commands to switch patterns. The webpage is hosted by a PC on the same network.

To bump the 3.3v signal voltage from the rPi up to 5v, I used this diode method.

Main Supplies:

  • Raspberry Pi A+
  • ws2812b LED strips
  • 200watt 5v supply
